## Skill behavior

- Name: `init-workflows` (slash: `/init-workflows`)
- Every invoke **audits** the checklist and **repairs gaps**
- Does **not** treat a marker file or one-line README as proof of readiness
- Overwrite policy:
  - Safe auto-repair: missing dirs, missing progress template (only if file absent), missing gitignore lines
  - Confirm before overwriting existing policy docs or AGENTS/CLAUDE content
- Greenfield: interactive setup (tracker → labels → domain), one decision at a time
- Existing policy docs: leave content; only create missing files
- AGENTS.md / CLAUDE.md: optional offer only — not required for “ready”

## Install (0.1)

## Skill package

Install unit is `skills/init-workflows/` only. **No hub path required at runtime.** Seeds, checklist, interview, and overwrite rules ship inside the skill.

Authored to Matt Pocock *writing-great-skills* levers: user-invoked, short description, steps with completion criteria, progressive disclosure to sibling files, single source of truth for templates, re-audit before READY.

Behavior summary:

- Every invoke **audits** then **repairs**; no marker-file readiness
- Safe auto-repair: gitignore, logs, missing progress only
- Confirm before writing policy docs
- Greenfield: one decision at a time
- AGENTS/CLAUDE: optional offer only

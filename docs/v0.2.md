# agent-workflows v0.2 — design freeze

Status: **0.2** — first consumer loop skill on top of v0.1 contracts. Not a host runner.

## Scope

| In 0.2 | Out of 0.2 |
|--------|------------|
| Skill package `loop-workflows` (user-invoked) | Host runner / `.ralph`-class shell / global CLI |
| Ralph-shaped **once** / explicit **max N** | Unbounded drain, Sandcastle |
| GitHub + `gh` only | Local-markdown loop, Linear as write target |
| Conservative PR lifecycle (no merge/close issue) | Stack / epic / Neon policies |
| Incomplete-claim resume; blocker soft-skip; skip-if non-draft PR | Dual implement/validate workers |
| **Implement quality** (typecheck, focused tests, broader pass before PR) | Ticket-format quality gates |
| init seed micro-update (Integration branch) | Bundling or forking Matt planning skills |
| README layers + companion guidance (no `docs/companions.md`) | Migrating full engineering skill set into hub |

## `loop-workflows` summary

| Topic | Freeze |
|-------|--------|
| Invocation | User-invoked only (`disable-model-invocation: true`) |
| Implement model | Same session orchestrates **and** implements (one issue per tick) |
| Default mode | **once** (`max_items=1`) |
| Multi-issue | Only with **explicit** positive N |
| Stops | COMPLETE (empty + no incomplete claim) · BLOCKED (none claimable) · MAX · HARD STOP |
| Tracker | GitHub only; hard stop otherwise |
| Pick | Oldest first; soft-skip open **Blocked by**, non-draft open PR, and **spec/PRD bodies** (Matt `/to-spec` shape — do not implement; re-label `ready-for-human`) |
| Preflight plan | Print **will resume #N** or **will pick #N** before claim; retry `gh` outside sandbox if keyring fails |
| Resume | Before new claim, finish at most one incomplete claim |
| Claim | Comment + remove `ready-for-agent` immediately |
| Success | Open PR (`Closes #N`), `ready-for-human`, no merge, no issue close |
| Quality | At implement: typecheck + focused tests + broader pass before PR; run any explicit commands listed on the issue |
| Git base | Optional **Integration branch** in `docs/agents/issue-tracker.md`; else default branch |
| Progress | `.agent-workflows/progress.md` (auto-create if missing) |
| Dirty tree | Hard stop |
| Empty queue | COMPLETE + soft companion hint (README/design; skill message) |
